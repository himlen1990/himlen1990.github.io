hexo备忘

node.js

install nvm
$ git clone https://github.com/creationix/nvm.git ~/.nvm
$ source ~/.nvm/nvm.sh

check nvm

$ nvm ls-remote
$ nvm install 0.12.7 (or other version)
$ node -v (check version)
$ nvm alias default v0.12.7

add to .bashrc
if [[ -s ~/.nvm/nvm.sh ]];
then source ~/.nvm/nvm.sh
fi

$ npm install -g hexo
$ hexo init blog
$ cd blog
$ npm install
for testing:
$ hexo server

create a github repository

open blog/_config.yml
deploy:
type: git
repository: xxxx.github.io.git
branch: master

$ npm install hexo-deployer-git –save
$ hexo deploy -g
then you can see your blog in (username).github.io

everytime you change:

$ hexo deploy -g or hexo d -g

change theme

copy the theme to your blog/themes file by using git clone xxxx theme/theme_name

then edit blog/_config.yml

theme: (theme name)
